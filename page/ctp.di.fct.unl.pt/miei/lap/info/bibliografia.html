Distribuição OCaml

A instalação do OCaml em máquinas Linux sempre foi trivial. Temos sorte, porque o sistema operativo "oficial" de LAP é um Ubuntu.

No caso do Windows e Mac, a instalação das versões mais recentes do OCaml tornou-se complicada. A versão 3.11 foi a última versão disponibilizada sob a forma executável para Windows e Mac, ou seja foi a última versão fácil de instalar. Assim, os alunos podem usar em casa a versão 3.11, mas no caso do 1º projeto devem ir aos laboratórios testar o seu programa antes de submeter a versão final.

  • OCaml para Debian ou Ubuntu: sudo apt-get install ocaml
  • OCaml 3.11.0 para Windows
  • OCaml 3.11.0 para MacOS X
  • OCaml distribution home (France)
